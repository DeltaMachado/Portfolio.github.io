Na disciplina de multimédia tive que realizar uma curta-metragem de imagem real para o projeto do 1º período, a baixo encontramos esse vídeo e uma breve explicação do mesmo.
"O Último Suspiro"
Este trabalho foi um projeto de grupo no âmbito da disciplina, depois de termos todas as cenas do filme gravadas passamos para uma fase de edição, usamos o programa Shotcut para editarmos a totalidade do filme. Com a parte do vídeo toda feita passamos para a parte sonora do mesmo. Finalmente fizemos um pequeno ajuste de coloração das cenas para o filme ficar mais coeso. Com a adição dos créditos demos a curta-metragem como concluída.
Nesta curta-metragem, tudo começa com um soldado que se prepara para visitar a campa do seu falecido irmão de armas, e mentaliza-se do que está por vir quando embreve este próprio partir em missão, reza pela sua segurança. Em campo de batalha encontra-se sozinho e tem de lutar para sobreviver, porém enquanto tenta encontrar o seu caminho, o inevitável acontece.
